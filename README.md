# Instruções de Instalação 

## INSTALANDO DEPENDENCIAS NODE
npm install

## INSTALANDO REACT HOOK FORM
npm install react-hook-form

## INSTALANDO MATERIAL UI
npm install @mui/material @emotion/react @emotion/styled @mui/icons-material

## CRIANDO BANCO E TABELAS
python src/Database/gerar_dados.py

## Obs: 
       O arquivo "instalarBiblioteca.bat" já faz essa instalação automaticamente, basta clicar duas vezes sobre ele.
       O arquivo "iniciarPrograma.bat" já roda o sistema, seguindo os mesmos passos anterior. (rodar apos a instalação das bibliotecas)
